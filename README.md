# Cuestionarios Oposicions Generalitat de Catalunya

Colección de cuestionarios interactivos para preparar las oposiciones de la Generalitat.

## 📱 Acceso desde el móvil (GitHub Pages)

Para acceder cómodamente desde el móvil activa **GitHub Pages** en este repositorio:

1. Ve a **Settings** → **Pages** (en la barra lateral izquierda).
2. En *Branch*, selecciona `main` y carpeta `/ (root)`.
3. Haz clic en **Save**.
4. En unos minutos la interfaz estará disponible en:

   **https://gottxxx.github.io/Oposiciones-Generalitat/**
